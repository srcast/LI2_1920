Detailed Description

Catálogo de Produtos.

Módulo de dados onde são guardados os códigos válidos de todos os produtos do ficheiro lido. Os códigos são organizados por índice alfabético, o que permite, de forma eficaz, saber quais e quantos são os produtos cujos códigos começam por uma determinada letra do alfabeto.
